#guia de tags HTML

TAGS GERAIS
<!doctype html> define tipo do documento(html5)
<html></html> engloba código html

<head></head> engloba partes não visíveis na página, scripts, etc
	head tags
	<title></title> define titulo da página
	<meta> define metadados usados na página como charset, etc

<body></body> engloba parte visível do site

TAGS DE CABECALHO E FORMATACAO DE TEXTO

<h1></h1> <h2></h2> tags de cabecalho, por padrao diminui tamanho da letra
para cada nivel de titulo, limite no h6

<hr> cria uma linha horizontal de separação
<p></p> paragrafo
<br> pula linha
<b></b> <strong></strong>(recomendável) negrito
<i></i> <em></em> (recomendável) itálico
<u></u> underline
<strike></strike> tachado

TAGS DE LISTA

<ul></ul> lista não ordenada, atributo type define o tipo de marcador
<ol></ol> lista ordenada, atributo type define o tipo de organizador

<li></li> item da lista

TAGS DE IMAGEM

<img src="">

TAGS DE LINK

<a href=""></a> âncora

TABELAS

<table border="px" width="px ou %"></table> tabela com atributos border e width de x px
<tr></tr> linha da tabela
<td></td> coluna
<th></th> cabeçalho

<td colspan="x"> define número de extensões das colunas
<tr rowspan="x"> define número de extensões das linhas

FORMS

<form action=""></form> tag de formulário com atributo de processamento 
<input type=""> tipo de entrada
